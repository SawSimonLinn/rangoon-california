# Background Music

To add background music to your restaurant website:

1. Add your music file here as `background-theme.mp3`
2. The file should be in MP3 format for best browser compatibility
3. Consider using ambient or instrumental music appropriate for a restaurant setting
4. Keep file size reasonable (under 5MB) for faster loading
5. Ensure you have proper licensing for any copyrighted music

## Suggested Music Types:

- Traditional Burmese instrumental music
- Ambient world music
- Soft acoustic tracks
- Nature sounds mixed with gentle melodies

## Free Music Resources:

- Freesound.org
- YouTube Audio Library
- Free Music Archive
- Incompetech (Kevin MacLeod)

The music player is currently configured to:

- Auto-play on the homepage (muted by default due to browser policies)
- Remember user's mute preference
- Show a volume control button in the bottom-right corner
